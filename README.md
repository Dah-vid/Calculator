# Calculator Project

A fully functional calculator built with vanilla JavaScript, HTML, and CSS. Final project for The Odin Project's Foundations course.

## Features

- ✅ Basic arithmetic operations (addition, subtraction, multiplication, division)
- ✅ Operator chaining (calculate results as you go: `12 + 7 - 5 = 14`)
- ✅ Multi-digit number support
- ✅ Decimal rounding to 8 places
- ✅ Error handling:
  - Divide by zero detection
  - Incomplete equation validation
- ✅ Clear button to reset calculator
- ✅ Space/hacker themed UI with green terminal aesthetic

## Technologies Used

- HTML5
- CSS3 (CSS Grid for layout)
- Vanilla JavaScript (no frameworks)

## What I Learned

- Managing application state with global variables
- Event delegation and DOM manipulation
- CSS Grid for responsive layouts
- Debugging with browser DevTools
- Git workflow and version control
